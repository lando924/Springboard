getElementById

    Selecting Elements in DOM
  • Selecting elements in the DOM is the first step to manipulating a web document. This is done through the document object, which contains properties and methods used to find and update elements, add events, and more. All methods for selecting elements are based in the document object and start with 'document dot something'.
  • The 'document.getElementById' method is used to retrieve an element based on its id. It finds the first matching element. With an id, there should only be one element with a given id on the page. If there were two elements with the same id, it would return the first one. The id is case-sensitive.
