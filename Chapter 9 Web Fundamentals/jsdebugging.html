    Common Errors
  • SyntaxError
  • ReferenceError
  • TypeError

Syntax Error: type something incorrect that JS doesn't understand. Example, 3 + ;4

ReferenceError: thrown when you try to access a variable that is not defined that means undefined Example: common in scope issue, scripts are out of order

TypeError:

  • trying to do something with a type that you can not
  • Accessing properties on undefined or null
  • Invoking ("calling") something that is not a function
