    • First Class Functions (Functions are Objects)
    • Functions in JS are first-class because of the following 3 features:
      1. functions can be passed as arguments to other functions
      2. the return value of a function can be another function
      3. function can be assigned to a variable
    • Anonymous Functions and Function Expressions
      1. Anonymous function is a function without a name
      2. function expression is a function that is assigned to a variable
      3. These types of functions can be passed as arguments, returned from a function, and stored in a variable.
    • Synchronous and Asynchronous Code
      1. Synchronous code is executed in sequence – each statement waits for the previous statement to finish before executing.
      2. Asynchronous code doesn't have to wait – your program can continue to run. You do this to keep your site or app responsive, reducing waiting time in situations where you can't predict the time something will take to complete.
    • setTimeout and setInterval Methods
      1. setTimeout and setInterval are built-in methods in JavaScript used for scheduling tasks.
      2. setTimeout allows you to schedule a task to run after a delay.
      3. setInterval allows you to schedule a task to run at regular intervals.

      4. Call Stack
          What happens in call stack.
        1. The call stack in JavaScript is essentially its to-do list. It's the structure where JavaScript keeps track of what it's been working on, what's incomplete, and what it needs to do next. Things are added to the call stack one by one and then removed as they are completed. It's like a task manager that JavaScript looks to, to figure out what it should do next.
      5. Anonymous Functions
        1. function that does not have a name. These functions are often used as arguments to other functions. They can be stored in variables or passed to other functions. They are called anonymous because they do not have a name of their own, and cannot be called again unless stored in a variable.
      6. Function Statements and Funtion Expressions
        1. A function statement in JavaScript is a standalone function with a name, defined using the function keyword. A function expression, on the other hand, is a function that is assigned to a variable. The function in a function expression can be anonymous, meaning it does not have a name of its own.
